# springboot-kubernates
Code for "SpringBoot + Kubernetes Tutorial"

# How to run?
$ git clone https://github.com/sreevani8/springboot-kubernetes.git
$ cd springboot-kubernetes
$ ./run.sh start
$ ./run.sh stop

$ ./run.sh start_infra
$ ./run.sh stop_infra

* To start only dependent services

$ ./run.sh start_infra
$ ./run.sh stop_infra
