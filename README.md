# Bookmarker-api
Code for "SpringBoot + Kubernates Tutorial"

# How to run?
$ git clone https://github.com/sreevani8/Bookmarker-api.git
$ cd Bookmarker-api
$ ./run.sh start
$ ./run.sh stop

$ ./run.sh start_infra
$ ./run.sh stop_infra

* To start only dependent services

$ ./run.sh start_infra
$ ./run.sh stop_infra
